# go-pkg

Repology But Free!

`go-pkg` is a command-line interface (CLI) tool designed to provide a comprehensive, open-source alternative for tracking, querying, and comparing package versions across multiple repository ecosystems directly from the terminal.

## Installation

### Go (Recommended)

To install the latest version of `go-pkg` using the Go toolchain, execute the following command:

```bash
go install github.com/grish-ka/go-pkg@latest
```
